Guia de CSS

Este guia contém exemplos práticos de como utilizar as principais propriedades e regras CSS. Cada seção aborda um conceito fundamental de estilo para a web.

1. Seletores CSS

Seletores Básicos

Os seletores são usados para selecionar elementos HTML e aplicar estilos. Os seletores básicos incluem seletores de tipo, classe e ID.

/* Seleciona todos os parágrafos */
p {
    color: blue;
}

/* Seleciona o elemento com a classe "header" */
.header {
    background-color: #3498db;
    color: white;
}

/* Seleciona o elemento com o ID "menu" */
#menu {
    font-size: 20px;
}

Este é um parágrafo com estilo aplicado.

Cabeçalho com classe .header
Este é um menu com ID #menu

Seletores Combinados

Os seletores combinados permitem aplicar estilos a elementos baseados em sua relação com outros elementos no documento.

/* Seleciona todos os  elementos dentro de um parágrafo */
p em {
    color: red;
}

/* Seleciona os links que são filhos diretos de um div */
div > a {
    text-decoration: none;
}

2. Propriedades de Texto

Cor e Fonte

Você pode alterar a cor, o tamanho, a família da fonte e o espaçamento entre letras usando CSS.

/* Definir cor e fonte */
h1 {
    color: #2980b9;
    font-family: 'Arial', sans-serif;
}

/* Alterar o espaçamento entre letras */
p {
    letter-spacing: 2px;
    font-size: 18px;
}

Título com cor e fonte personalizada

3. Propriedades de Cor

Definindo Cores

CSS permite definir cores usando nomes de cores, valores hexadecimais, RGB ou HSL.

/* Cor usando nome */
p {
    color: green;
}

/* Cor usando hexadecimal */
div {
    background-color: #3498db;
}

/* Cor usando RGB */
h1 {
    color: rgb(255, 0, 0);
}

/* Cor usando HSL */
button {
    background-color: hsl(120, 100%, 50%);
}

4. Layout e Box Model

Box Model

O box model em CSS envolve as propriedades de margin, border, padding e o conteúdo do elemento.

/* Box model */
div {
    width: 200px;
    padding: 20px;
    border: 5px solid #3498db;
    margin: 10px;
}
Elemento com box model aplicado

Layouts com Flexbox

Flexbox é um método moderno de criação de layouts flexíveis.

/* Flexbox container */
.container {
    display: flex;
    justify-content: space-between;
}

.item {
    background-color: #3498db;
    padding: 20px;
    color: white;
}

5. Propriedades de Imagem e Fundo

Imagens e Fundos

Você pode definir imagens como fundo e controlar sua exibição com CSS.

/* Definir imagem de fundo */
div {
    background-image: url('https://via.placeholder.com/200');
    background-size: cover;
    width: 200px;
    height: 200px;
}

6. Propriedades de Posição

Posicionamento Relativo e Absoluto

CSS oferece várias maneiras de posicionar elementos: estático, relativo, absoluto, fixo, e sticky.

/* Posicionamento absoluto */
div {
    position: absolute;
    top: 50px;
    left: 100px;
}

/* Posicionamento relativo */
p {
    position: relative;
    top: 10px;
    left: 20px;
}

7. Transições e Animações

Transições

Transições em CSS permitem animar mudanças de estilo de forma suave.

/* Transição de cor */
button {
    background-color: #3498db;
    transition: background-color 0.5s ease;
}

button:hover {
    background-color: #2ecc71;
}

Animações

As animações em CSS permitem criar sequências mais complexas de transformações visuais.

/* Animação de rotação */
@keyframes rotacao {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }
}

div {
    animation: rotacao 2s infinite linear;
}

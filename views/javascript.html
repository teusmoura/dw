Guia de JavaScript para Iniciantes

Este guia apresenta os conceitos básicos de JavaScript, incluindo variáveis, funções, eventos e interação com o DOM, para ajudar iniciantes a começar a desenvolver páginas web interativas.

1. O que é JavaScript?

JavaScript é uma linguagem de programação usada para adicionar interatividade a páginas web. Diferente de HTML e CSS, que controlam a estrutura e o estilo de uma página, o JavaScript permite que você faça a página "responder" às ações do usuário, como cliques e digitação.

2. Variáveis e Tipos de Dados

Em JavaScript, variáveis são usadas para armazenar dados, como números e textos. Você pode usar var, let ou const para declarar variáveis, sendo let e const as opções mais modernas.

Exemplo de Declaração de Variáveis

// Usando let para criar uma variável que pode mudar
let idade = 25;

// Usando const para criar uma variável constante
const nome = "João";

Tipos de Dados Comuns

  • String: Cadeia de caracteres, como "Olá, mundo!"
  • Number: Números, como 100 ou 3.14
  • Boolean: Verdadeiro ou falso (true ou false)
let nome = "Maria"; // String
let idade = 30; // Number
let ativo = true; // Boolean

3. Funções

Uma função é um bloco de código que pode ser executado sempre que necessário. Elas ajudam a organizar o código e reutilizar lógica sem precisar repetí-la.

Exemplo de Função

// Definindo uma função que exibe uma mensagem
function saudacao() {
    console.log("Olá, bem-vindo!");
}

// Chamando a função
saudacao();

Funções também podem receber parâmetros e retornar valores.

Função com Parâmetros e Retorno

function soma(a, b) {
    return a + b;
}

let resultado = soma(5, 10);
console.log(resultado); // Exibe 15

4. Condicionais

As condicionais permitem que você execute diferentes blocos de código dependendo de certas condições. A estrutura mais comum é o if e else.

Exemplo de Condicional

let idade = 18;

if (idade >= 18) {
    console.log("Você é maior de idade.");
} else {
    console.log("Você é menor de idade.");
}

5. Loops

Os loops permitem que você repita um bloco de código várias vezes. O loop for é o mais comum.

Exemplo de Loop

for (let i = 0; i < 5; i++) {
    console.log("O valor de i é " + i);
}

Outro tipo de loop é o while, que continua executando enquanto a condição for verdadeira.

Exemplo de Loop While

let i = 0;

while (i < 5) {
    console.log(i);
    i++;
}

6. Interação com o DOM

O DOM (Document Object Model) é a representação da página web em JavaScript. Com ele, você pode acessar e modificar os elementos da página.

Selecionando Elementos

// Seleciona um elemento pelo ID
let titulo = document.getElementById("titulo");

// Modifica o texto do elemento
titulo.textContent = "Novo título";

Você também pode reagir a eventos como cliques, usando JavaScript para fazer a página interativa.

Exemplo de Evento de Clique

// Seleciona o botão
let botao = document.getElementById("meuBotao");

// Adiciona um evento de clique
botao.addEventListener("click", function() {
    alert("Botão clicado!");
});

7. Arrays e Objetos

Arrays

Arrays são listas de valores. Você pode armazenar vários dados em uma única variável.

Exemplo de Array

let frutas = ["Maçã", "Banana", "Laranja"];
console.log(frutas[0]); // Maçã

Objetos

Objetos são usados para armazenar dados em pares de chave e valor. Eles são como dicionários.

Exemplo de Objeto

let pessoa = {
    nome: "Carlos",
    idade: 28,
    cidade: "São Paulo"
};

console.log(pessoa.nome); // Carlos

8. Comandos Úteis no JavaScript

console.log() - Exibir no Console

Usado para exibir mensagens e valores no console do navegador. É muito útil para depuração.

console.log("Olá, mundo!");

setTimeout() - Atrasar Execução

Executa uma função depois de um certo tempo (em milissegundos).

setTimeout(function() {
    console.log("Esta mensagem aparece após 2 segundos.");
}, 2000);

setInterval() - Repetir Execução

Executa uma função repetidamente a cada intervalo de tempo.

setInterval(function() {
    console.log("Esta mensagem aparece a cada 1 segundo.");
}, 1000);

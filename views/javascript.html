JavaScript Completo

Este guia apresenta uma visão completa de JavaScript, cobrindo desde variáveis, funções e condicionais até a interação avançada com o DOM e manipulação de eventos. Prepare-se para criar páginas interativas e dinâmicas com vários exemplos práticos.

1. O que é JavaScript?

JavaScript é uma linguagem de programação usada para adicionar interatividade a páginas web. Diferente de HTML e CSS, que controlam a estrutura e o estilo de uma página, o JavaScript permite que você faça a página "responder" às ações do usuário, como cliques e digitação.

2. Variáveis e Tipos de Dados

Em JavaScript, variáveis são usadas para armazenar dados, como números, textos e valores booleanos. Podemos usar var, let ou const para declarar variáveis.

Exemplo de Declaração de Variáveis

let nome = "João"; // String
const idade = 25; // Number
let ativo = true; // Boolean

Tipos de Dados Comuns

  • String: Cadeias de caracteres, como "Olá, mundo!"
  • Number: Números, como 100 ou 3.14
  • Boolean: Valores verdadeiros ou falsos (true ou false)
  • Array: Lista de valores, como ["Maçã", "Banana", "Laranja"]
  • Object: Coleção de pares chave-valor, como { nome: "Carlos", idade: 30 }

3. Funções

Funções são blocos de código que podem ser executados sempre que necessário. Elas ajudam a organizar o código e reutilizar lógica sem precisar repetí-la.

Exemplo de Função Básica

function saudacao() {
    console.log("Olá, bem-vindo!");
}

saudacao();

Função com Parâmetros e Retorno

function soma(a, b) {
    return a + b;
}

let resultado = soma(10, 20);
console.log(resultado); // Exibe 30

4. Condicionais

As condicionais permitem que você execute diferentes blocos de código dependendo de certas condições. A estrutura mais comum é o if, else if e else.

Exemplo de Condicional

let idade = 18;

if (idade >= 18) {
    console.log("Você é maior de idade.");
} else {
    console.log("Você é menor de idade.");
}

Condicional com Vários Ramos

let nota = 85;

if (nota >= 90) {
    console.log("Aprovado com distinção");
} else if (nota >= 60) {
    console.log("Aprovado");
} else {
    console.log("Reprovado");
}

5. Loops

Loops permitem que você repita um bloco de código várias vezes. O loop for é o mais comum.

Exemplo de Loop For

for (let i = 0; i < 5; i++) {
    console.log("O valor de i é " + i);
}

Exemplo de Loop While

let i = 0;

while (i < 5) {
    console.log(i);
    i++;
}

6. Interação com o DOM

O DOM (Document Object Model) é a estrutura hierárquica da página web. Com JavaScript, podemos acessar, modificar e manipular elementos do DOM dinamicamente, criando interações avançadas para usuários.

Selecionando e Modificando Elementos

Podemos selecionar elementos da página por ID, classe ou tag, e alterar suas propriedades como o texto ou estilos.

Este texto será alterado dinamicamente.

Exemplo Prático de Seleção e Modificação

// Seleciona o elemento pelo ID
let elemento = document.getElementById("resultadoInteracao");

// Modifica o conteúdo do elemento ao clicar no botão
document.getElementById("meuBotao").addEventListener("click", function() {
    elemento.textContent = "O conteúdo foi modificado!";
    elemento.classList.add("highlight");
});

Adicionar Novos Elementos ao DOM

Além de modificar, você também pode criar novos elementos dinamicamente e adicioná-los à página.

// Função para adicionar novos elementos
document.getElementById("botaoAdicionar").addEventListener("click", function() {
    let novoParagrafo = document.createElement("p");
    novoParagrafo.textContent = "Este parágrafo foi adicionado dinamicamente!";
    document.getElementById("output").appendChild(novoParagrafo);
});

7. Arrays e Objetos

Arrays

Arrays são usados para armazenar múltiplos valores em uma única variável. Eles são úteis para listas de itens.

Exemplo de Array

let frutas = ["Maçã", "Banana", "Laranja"];
console.log(frutas[0]); // Exibe "Maçã"

Objetos

Objetos são coleções de pares chave-valor que permitem armazenar dados estruturados.

Exemplo de Objeto

let pessoa = {
    nome: "Carlos",
    idade: 28,
    cidade: "São Paulo"
};

console.log(pessoa.nome); // Exibe "Carlos"

Exemplo de Manipulação de Arrays e Objetos

// Adicionando e removendo itens de um array
let carros = ["Fusca", "Gol", "Civic"];
carros.push("Palio"); // Adiciona um item ao final
carros.pop(); // Remove o último item

// Acessando propriedades de um objeto
let aluno = {
    nome: "Ana",
    idade: 22,
    curso: "Engenharia"
};

aluno.curso = "Medicina"; // Modifica o valor de uma propriedade
console.log(aluno.curso); // Exibe "Medicina"

8. Comandos Úteis no JavaScript

console.log() - Exibir no Console

Usado para exibir mensagens e valores no console do navegador. É muito útil para depuração.

console.log("Olá, mundo!");

setTimeout() - Atrasar Execução

Executa uma função depois de um certo tempo (em milissegundos).

setTimeout(function() {
    console.log("Esta mensagem aparece após 2 segundos.");
}, 2000);

setInterval() - Repetir Execução

Executa uma função repetidamente a cada intervalo de tempo.

setInterval(function() {
    console.log("Esta mensagem aparece a cada 1 segundo.");
}, 1000);

9. Eventos e Interatividade

Eventos são ações que ocorrem na página, como cliques, digitação ou rolagem. Podemos usar JavaScript para responder a esses eventos e criar interações.

Eventos de Clique

Vamos adicionar um evento de clique a um botão:

document.getElementById("meuBotaoEvento").addEventListener("click", function() {
    alert("Você clicou no botão!");
});

Eventos de Teclado

Também podemos capturar eventos de teclado para realizar ações específicas.

document.getElementById("campoTexto").addEventListener("keydown", function(event) {
    console.log("Você pressionou a tecla: " + event.key);
});

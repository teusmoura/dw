JavaScript Completo

Este guia apresenta uma visão completa de JavaScript, cobrindo desde variáveis, funções e condicionais até a interação avançada com o DOM e manipulação de eventos. Prepare-se para criar páginas interativas e dinâmicas com vários exemplos práticos.

1. Elementos Passíveis de Manipulação com JavaScript

Com JavaScript, você pode manipular quase todos os elementos da página. Abaixo estão alguns dos mais comuns:

  • **Texto**: Cabeçalhos (<h1>, <h2>), parágrafos (<p>), e elementos <span>.
  • **Imagens**: <img>, alterando src ou atributos de estilo.
  • **Botões**: <button>, eventos de clique e mudança de aparência.
  • **Inputs**: Campos de texto, senhas, checkboxes, radio buttons, entre outros.
  • **Listas**: <ul>, <ol> e suas respectivas <li>.
  • **Divs**: Contêineres gerais com IDs e classes específicas.
  • **Links**: <a>, alterando o href e o comportamento.

2. Manipulação de Texto

Podemos manipular texto de cabeçalhos, parágrafos e elementos de texto embutidos como <span>.

Exemplo de Manipulação de Cabeçalhos

Título Original

// Manipulando o texto de um cabeçalho
document.getElementById("alterarTitulo").addEventListener("click", function() {
    let titulo = document.getElementById("tituloPrincipal");
    titulo.textContent = "Título Modificado com Sucesso!";
});

Manipulando Texto de Parágrafos

Este é um parágrafo de exemplo.

// Alterar texto de um parágrafo
document.getElementById("alterarParagrafo").addEventListener("click", function() {
    let paragrafo = document.getElementById("meuParagrafo");
    paragrafo.textContent = "O texto foi alterado dinamicamente!";
});

3. Manipulação de Imagens

Podemos alterar a fonte da imagem (src) ou manipular suas propriedades de estilo.

Exemplo de Alteração de Imagem

// Alterar a fonte da imagem
document.getElementById("alterarImagem").addEventListener("click", function() {
    let imagem = document.getElementById("minhaImagem");
    imagem.src = "https://via.placeholder.com/300";
});

Alteração de Estilos de Imagem

// Alterar o estilo da imagem
document.getElementById("alterarEstiloImagem").addEventListener("click", function() {
    let imagem = document.getElementById("minhaImagem");
    imagem.style.borderRadius = "50%";
    imagem.style.boxShadow = "0px 0px 10px #000";
});

4. Manipulação de Botões

Podemos manipular botões adicionando eventos de clique e alterando seu estilo.

Exemplo de Botão com Estilo Dinâmico

// Alterar estilo de um botão ao clicar
document.getElementById("botaoEstilizado").addEventListener("click", function() {
    let botao = document.getElementById("botaoEstilizado");
    botao.style.backgroundColor = "#28a745";
    botao.style.color = "#fff";
    botao.textContent = "Clicado!";
});

5. Manipulação de Inputs

Podemos capturar e alterar valores de diferentes tipos de inputs: texto, senha, radio buttons, checkboxes, etc.

Exemplo de Captura de Valores

// Captura o valor de um campo de texto
document.getElementById("capturarInput").addEventListener("click", function() {
    let nome = document.getElementById("meuInput").value;
    alert("Nome digitado: " + nome);
});

6. Manipulação de Links

Podemos alterar o destino de um link (href) ou seu texto dinamicamente.

Exemplo de Manipulação de Link

Visite o Site 
// Alterar o href e o texto de um link
document.getElementById("alterarLink").addEventListener("click", function() {
    let link = document.getElementById("meuLink");
    link.href = "https://www.google.com";
    link.textContent = "Visite o Google";
});

7. Manipulação de Listas

Podemos adicionar, remover ou alterar itens em listas não ordenadas (<ul>) ou ordenadas (<ol>).

Exemplo de Adição de Item a uma Lista

  • Item 1
  • Item 2
  • Item 3
// Adiciona um novo item à lista
document.getElementById("adicionarItemLista").addEventListener("click", function() {
    let lista = document.getElementById("minhaLista");
    let novoItem = document.createElement("li");
    novoItem.textContent = "Novo Item";
    lista.appendChild(novoItem);
});

8. Manipulação de Divs

Podemos alterar o conteúdo de <div>s ou seus estilos e adicionar novos contêineres à página.

Exemplo de Manipulação de Div

Conteúdo Original da Div
// Alterar o conteúdo de uma div
document.getElementById("alterarDiv").addEventListener("click", function() {
    let div = document.getElementById("divPrincipal");
    div.textContent = "O conteúdo da div foi alterado!";
    div.classList.add("highlight");
});

JavaScript Completo

Este guia apresenta uma visão completa de JavaScript, cobrindo desde variáveis, funções e condicionais até a interação avançada com o DOM e manipulação de eventos. Prepare-se para criar páginas interativas e dinâmicas com vários exemplos práticos.

1. Elementos Passíveis de Manipulação com JavaScript

Com JavaScript, você pode manipular quase todos os elementos da página. Abaixo estão alguns dos mais comuns:

  • **Texto**: Cabeçalhos (<h1>, <h2>), parágrafos (<p>), e elementos <span>.
  • **Imagens**: <img>, alterando src ou atributos de estilo.
  • **Botões**: <button>, eventos de clique e mudança de aparência.
  • **Inputs**: Campos de texto, senhas, checkboxes, radio buttons, entre outros.
  • **Listas**: <ul>, <ol> e suas respectivas <li>.
  • **Divs**: Contêineres gerais com IDs e classes específicas.
  • **Links**: <a>, alterando o href e o comportamento.

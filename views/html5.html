HTML

Este guia detalhado contém exemplos práticos e explicações para as principais tags HTML. Cada seção apresenta o uso e a visualização das tags.

1. Estrutura e Metadados

<html> e <head>

Define a estrutura básica do documento HTML e o cabeçalho da página, respectivamente.

<html lang="pt-br">
<head>
    <!-- Metadados, título e links para estilos -->
</head>
<body>
    <!-- Conteúdo principal do documento -->
</body>
</html>
[Não há resultado visual específico]

<meta> e <title>

Define metadados da página e o título exibido na aba do navegador.

<meta charset="UTF-8">
<title>Guia Completo de Tags HTML</title>
[Título exibido na aba do navegador]

2. Estruturação de Conteúdo

<header>, <nav>, <section>, <article>, <aside>, <footer> e <div>

Essas tags ajudam a organizar o conteúdo da página em blocos semânticos.

<header>Cabeçalho do site</header>
<nav>Navegação principal</nav>
<section>Seção de conteúdo</section>
<article>Artigo ou post específico</article>
<aside>Barra lateral ou conteúdo secundário</aside>
<footer>Rodapé da página</footer>
<div>Divisão genérica do conteúdo</div>

Cabeçalho do site
Navegação principal

Seção de conteúdo

Artigo ou post específico

Barra lateral ou conteúdo secundário

Rodapé da página

Divisão genérica do conteúdo

3. Formatação de Texto

<p>, <h1> a <h6>, <strong>, <em>, <u>, <br>

Essas tags definem parágrafos, títulos, negrito, itálico, sublinhado e quebras de linha.

<p>Este é um parágrafo.</p>
<h1>Título de Nível 1</h1>
<h2>Título de Nível 2</h2>
<strong>Texto em negrito</strong>
<em>Texto em itálico</em>
<u>Texto sublinhado</u>
<br>Quebra de linha

Este é um parágrafo.

Título de Nível 1

Título de Nível 2

Texto em negrito
Texto em itálico
Texto sublinhado
Quebra de linha

<blockquote> e <pre>

Estas tags são usadas para citações em bloco e blocos de código pré-formatados.

<blockquote>Isto é uma citação.</blockquote>
<pre>
    <code>Código pré-formatado</code>
</pre>
Isto é uma citação.
Código pré-formatado

4. Tipos de Input

<input> - Vários tipos de entrada

Os inputs podem ter vários tipos, como texto, email, senha, número, data, entre outros.

<form>
    <input type="text" placeholder="Digite seu nome">
    <br>
    <input type="email" placeholder="Digite seu email">
    <br>
    <input type="password" placeholder="Digite sua senha">
    <br>
    <input type="number" placeholder="Digite sua idade">
    <br>
    <input type="date" placeholder="Selecione uma data">
    <br>
    <input type="checkbox"> Concordo com os termos
    <br>
    <input type="radio" name="genero" value="masculino"> Masculino
    <br>
    <input type="radio" name="genero" value="feminino"> Feminino
    <br>
    <input type="range" min="0" max="100"> Volume
    <br>
    <button type="submit">Enviar</button>
</form>

Feminino

5. Multimídia

<audio>, <video>, <source>, <track>

Essas tags são usadas para adicionar áudio e vídeo ao documento HTML.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Seu navegador não suporta o elemento de áudio.
</audio>

<video controls>
    <source src="video.mp4" type="video/mp4">
    Seu navegador não suporta o elemento de vídeo.
</video>

6. Tabelas

<table>, <tr>, <th>, <td>

Cria tabelas com linhas (<tr>), cabeçalhos (<th>) e células (<td>).

<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Celula 1</td>
        <td>Celula 2</td>
    </tr>
</table>
Cabeçalho 1 Cabeçalho 2
Celula 1 Celula 2

7. Elementos Interativos e Gráficos

<details>, <summary>, <dialog>, <canvas>, <svg>

Esses elementos permitem interatividade e gráficos em HTML.

<details>
    <summary>Clique para expandir</summary>
    Aqui está o conteúdo expandido.
</details>

<canvas id="meuCanvas">Seu navegador não suporta o elemento canvas.</canvas>
Clique para expandir Aqui está o conteúdo expandido.

Seu navegador não suporta o elemento canvas.

8. Acessibilidade

<img> com atributo alt e atributos aria

O atributo alt em imagens e os atributos aria melhoram a acessibilidade para leitores de tela e outros dispositivos de assistência.

<img src="imagem.jpg" alt="Descrição da imagem para leitores de tela">

<button aria-label="Fechar janela">X</button>

9. Elementos Sem Fechamento

<br>, <hr>, <img>, <meta>

Essas tags não possuem uma tag de fechamento e são usadas para quebra de linha, linha horizontal, imagens e metadados.

<br> 
<hr> 
<img src="imagem.jpg" alt="Imagem descritiva"> 
<meta name="description" content="Descrição da página">
[Exemplo: Quebra de linha e linha horizontal não visíveis aqui]

10. Elementos de Código

<code> e <pre>

Cria blocos de código e texto pré-formatado.

<code>document.write('Hello, World!');</code>
<pre>
    function hello() {
        console.log('Hello, World!');
    }
</pre>
document.write('Hello, World!');

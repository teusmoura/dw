Guia Completo de JSON

Este guia cobre o conceito de JSON (JavaScript Object Notation), como utilizá-lo em JavaScript e exemplos práticos de como converter dados de e para JSON.

1. O que é JSON?

JSON (JavaScript Object Notation) é um formato leve de intercâmbio de dados que é fácil para os humanos lerem e escreverem, e fácil para as máquinas interpretarem e gerarem. Ele é amplamente utilizado para enviar e receber dados entre um servidor e um cliente na web.

O formato JSON é baseado em duas estruturas principais:

  • **Coleções de pares chave/valor**: semelhantes a objetos em JavaScript.
  • **Listas ordenadas de valores**: semelhantes a arrays em JavaScript.

2. Exemplo de Estrutura JSON

Abaixo está um exemplo de um objeto JSON que representa uma pessoa com propriedades como nome, idade, e uma lista de hobbies:

{
    "nome": "Carlos",
    "idade": 30,
    "email": "carlos@email.com",
    "hobbies": ["leitura", "natação", "fotografia"]
}

O formato JSON usa chaves duplas (") para os nomes de propriedades e valores do tipo string. Números e valores booleanos (true/false) são escritos sem aspas.

3. Convertendo entre JSON e Objetos JavaScript

Em JavaScript, podemos converter objetos para o formato JSON (serializar) e converter JSON de volta para objetos JavaScript (desserializar) usando dois métodos principais: JSON.stringify() e JSON.parse().

3.1 Convertendo um Objeto para JSON

let pessoa = {
    nome: "Maria",
    idade: 25,
    email: "maria@email.com"
};

// Converte o objeto JavaScript em uma string JSON
let jsonPessoa = JSON.stringify(pessoa);
console.log(jsonPessoa); // {"nome":"Maria","idade":25,"email":"maria@email.com"}

3.2 Convertendo JSON para Objeto

let json = '{"nome":"João", "idade":35, "email":"joao@email.com"}';

// Converte a string JSON para um objeto JavaScript
let objetoPessoa = JSON.parse(json);
console.log(objetoPessoa.nome); // João

4. Enviando e Recebendo JSON com fetch

Ao fazer uma requisição para um servidor, podemos enviar e receber dados no formato JSON. A API fetch permite trabalhar facilmente com JSON em requisições HTTP.

4.1 Enviando JSON

let dados = {
    nome: "João",
    email: "joao@email.com"
};

// Fazendo uma requisição POST e enviando dados JSON
fetch("https://exemplo.com/api", {
    method: "POST",
    headers: {
        "Content-Type": "application/json"
    },
    body: JSON.stringify(dados)
})
.then(response => response.json())
.then(data => console.log("Sucesso:", data))
.catch(error => console.error("Erro:", error));

4.2 Recebendo JSON

// Fazendo uma requisição GET e recebendo dados JSON
fetch("https://exemplo.com/api/dados")
    .then(response => response.json())
    .then(data => {
        console.log(data); // Exibe o JSON recebido
    })
    .catch(error => console.error("Erro:", error));

5. Manipulação de JSON Complexos

JSON pode representar dados complexos, como arrays aninhados e objetos dentro de objetos. Abaixo, veremos como manipular esses dados.

5.1 Acessando Propriedades de Objetos Aninhados

let pessoa = {
    nome: "Ana",
    endereco: {
        cidade: "São Paulo",
        estado: "SP"
    }
};

// Acessando a propriedade "cidade" dentro do objeto "endereco"
console.log(pessoa.endereco.cidade); // São Paulo

5.2 Iterando sobre Arrays em JSON

let dados = {
    nome: "Carlos",
    hobbies: ["futebol", "guitarra", "jogos"]
};

// Itera sobre o array de hobbies
dados.hobbies.forEach(function(hobby) {
    console.log(hobby);
});
// Saída:
// futebol
// guitarra
// jogos

6. Armazenando JSON no Local Storage

O Local Storage permite que você armazene dados localmente no navegador do usuário. Como o Local Storage só pode armazenar strings, você pode usar JSON para converter e armazenar objetos complexos.

Armazenando JSON no Local Storage

// Objeto que será armazenado
let usuario = {
    nome: "Pedro",
    idade: 40
};

// Converter o objeto em JSON e armazenar no Local Storage
localStorage.setItem("usuario", JSON.stringify(usuario));

Recuperando JSON do Local Storage

// Recuperar e converter o JSON para um objeto JavaScript
let usuarioSalvo = localStorage.getItem("usuario");
let objetoUsuario = JSON.parse(usuarioSalvo);

console.log(objetoUsuario.nome); // Pedro

7. Boas Práticas com JSON

Ao trabalhar com JSON, algumas boas práticas devem ser seguidas para garantir a segurança e a legibilidade do código:

  • **Sempre validar JSON recebido de fontes externas** para evitar problemas de segurança.
  • **Utilize sempre o tipo correto de dados**, como números, strings, booleanos, etc.
  • **Não armazene dados sensíveis no Local Storage**, já que ele pode ser acessado por qualquer script na página.

8. Comandos Úteis para JSON

JSON.stringify() - Converter Objeto em JSON

let objeto = { nome: "Lucas", idade: 28 };
let jsonString = JSON.stringify(objeto);

JSON.parse() - Converter JSON em Objeto

let jsonString = '{"nome": "Lucas", "idade": 28}';
let objeto = JSON.parse(jsonString);

fetch() - Trabalhando com Requisições JSON

fetch("https://api.exemplo.com/dados")
    .then(response => response.json())
    .then(data => console.log(data));

localStorage.setItem() e localStorage.getItem() - Armazenar e Recuperar JSON

localStorage.setItem("usuario", JSON.stringify(usuario));
let usuario = JSON.parse(localStorage.getItem("usuario"));

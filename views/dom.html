DOM com JavaScript

Este guia explica o que é o DOM e mostra várias maneiras de manipulá-lo usando JavaScript, com exemplos práticos para ajudar você a interagir com os elementos de uma página web.

1. O que é o DOM?

O DOM (Document Object Model) é uma representação da estrutura de uma página HTML ou XML. Ele transforma a página em um modelo de objetos que podem ser manipulados usando JavaScript. Com o DOM, você pode alterar o conteúdo, a estrutura e o estilo de uma página web de forma dinâmica.

2. Selecionando Elementos no DOM

Para manipular o DOM, o primeiro passo é selecionar os elementos que você deseja modificar. Existem várias maneiras de fazer isso usando JavaScript.

2.1 Selecionando Elementos por ID

// Seleciona um elemento pelo ID
let elemento = document.getElementById("meuElemento");

2.2 Selecionando Elementos por Classe

// Seleciona todos os elementos com a classe "classeExemplo"
let elementos = document.getElementsByClassName("classeExemplo");

2.3 Selecionando Elementos por Tag

// Seleciona todos os elementos 

let paragrafos = document.getElementsByTagName("p");

2.4 Selecionando com Query Selector

querySelector permite selecionar o primeiro elemento que corresponde a um seletor CSS, enquanto querySelectorAll seleciona todos os elementos que correspondem ao seletor.

// Seleciona o primeiro elemento com a classe .meuElemento
let elemento = document.querySelector(".meuElemento");

// Seleciona todos os elementos 

let divs = document.querySelectorAll("div");

3. Modificando Conteúdo e Estilo

Depois de selecionar os elementos, você pode alterar o conteúdo ou o estilo deles dinamicamente.

3.1 Modificando Texto

// Seleciona um elemento e altera o seu texto
let titulo = document.getElementById("titulo");
titulo.textContent = "Novo título";

3.2 Modificando HTML Interno

// Modifica o HTML interno de um elemento
let container = document.getElementById("container");
container.innerHTML = "<p>Este é um novo parágrafo.</p>";

3.3 Alterando o Estilo de um Elemento

// Altera o estilo CSS de um elemento
let paragrafo = document.querySelector("p");
paragrafo.style.color = "blue";
paragrafo.style.fontSize = "20px";

4. Criando e Removendo Elementos

Além de modificar elementos existentes, você pode criar novos elementos ou remover elementos da página.

4.1 Criando um Novo Elemento

// Cria um novo elemento 
 e adiciona à página
let novoDiv = document.createElement("div");
novoDiv.textContent = "Este é um novo div";
document.body.appendChild(novoDiv);

4.2 Removendo um Elemento

// Remove um elemento da página
let elementoParaRemover = document.getElementById("meuElemento");
elementoParaRemover.remove();

5. Manipulação de Atributos

Você pode modificar atributos de elementos, como o src de uma imagem ou o href de um link.

5.1 Modificando Atributos

// Altera o atributo src de uma imagem
let imagem = document.getElementById("minhaImagem");
imagem.setAttribute("src", "nova-imagem.jpg");

5.2 Obtendo Atributos

// Obtém o valor de um atributo
let link = document.querySelector("a");
let href = link.getAttribute("href");
console.log(href);

5.3 Removendo Atributos

// Remove um atributo de um elemento
let input = document.getElementById("meuInput");
input.removeAttribute("disabled");

6. Trabalhando com Eventos

Eventos permitem que você adicione interatividade às páginas, como responder a cliques, teclas pressionadas e outros tipos de interações do usuário.

6.1 Adicionando um Evento de Clique

// Seleciona o botão e adiciona um evento de clique
let botao = document.getElementById("botao");
botao.addEventListener("click", function() {
    alert("Botão clicado!");
});

6.2 Evento de Tecla Pressionada

// Executa uma função quando uma tecla é pressionada
document.addEventListener("keydown", function(event) {
    console.log("Tecla pressionada: " + event.key);
});

6.3 Removendo Eventos

// Função de evento
function alerta() {
    alert("Evento disparado!");
}

// Adiciona o evento
botao.addEventListener("click", alerta);

// Remove o evento
botao.removeEventListener("click", alerta);

7. Navegando no DOM

Você pode navegar entre os nós de elementos, permitindo acessar elementos "parent" (pai), "child" (filho), ou "siblings" (irmãos).

7.1 Acessando o Elemento Pai

// Acessa o elemento pai de um parágrafo
let paragrafo = document.querySelector("p");
let pai = paragrafo.parentElement;
console.log(pai);

7.2 Acessando os Elementos Filhos

// Acessa todos os filhos de um div
let div = document.querySelector("div");
let filhos = div.children;
console.log(filhos);

7.3 Acessando o Próximo Irmão

// Acessa o próximo irmão de um elemento
let primeiroItem = document.querySelector("li");
let proximoItem = primeiroItem.nextElementSibling;
console.log(proximoItem);

8. Comandos Úteis para Manipulação do DOM

document.createElement() - Criar um Elemento

Cria um novo elemento HTML, que pode ser adicionado ao DOM.

let novoParagrafo = document.createElement("p");
novoParagrafo.textContent = "Este é um novo parágrafo";
document.body.appendChild(novoParagrafo);

element.remove() - Remover um Elemento

Remove um elemento do DOM.

let elemento = document.getElementById("meuElemento");
elemento.remove();

element.setAttribute() - Modificar Atributos

Modifica ou define um novo atributo para o elemento.

let link = document.querySelector("a");
link.setAttribute("href", "https://www.example.com");

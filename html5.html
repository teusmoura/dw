Guia Completo de Tags HTML

Este guia detalhado contém exemplos práticos e explicações para as principais tags HTML. Cada seção apresenta o uso e a visualização das tags.

1. Estrutura e Metadados

<html> e <head>

Define a estrutura básica do documento HTML e o cabeçalho da página, respectivamente.

<html lang="pt-br">
<head>
    <!-- Metadados, título e links para estilos -->
</head>
<body>
    <!-- Conteúdo principal do documento -->
</body>
</html>
[Não há resultado visual específico]

<meta> e <title>

Define metadados da página e o título exibido na aba do navegador.

<meta charset="UTF-8">
<title>Guia Completo de Tags HTML</title>
[Título exibido na aba do navegador]

<style> e <script>

Define estilos internos e scripts JavaScript, respectivamente.

<style>
    body { font-family: Arial, sans-serif; }
</style>
<script>
    console.log('Script carregado.');
</script>
[Estilos e scripts são executados, mas não têm resultado visual direto]

2. Estruturação de Conteúdo

<header>, <nav>, <section>, <article>, <aside>, <footer> e <div>

Essas tags ajudam a organizar o conteúdo da página em blocos semânticos.

<header>Cabeçalho do site</header>
<nav>Navegação principal</nav>
<section>Seção de conteúdo</section>
<article>Artigo ou post específico</article>
<aside>Barra lateral ou conteúdo secundário</aside>
<footer>Rodapé da página</footer>
<div>Divisão genérica do conteúdo</div>

3. Formatação de Texto

<p>, <h1> a <h6>, <strong>, <em>, <u>, <br>

Essas tags definem parágrafos, títulos, negrito, itálico, sublinhado e quebras de linha.

<p>Este é um parágrafo.</p>
<h1>Título de Nível 1</h1>
<h2>Título de Nível 2</h2>
<strong>Texto em negrito</strong>
<em>Texto em itálico</em>
<u>Texto sublinhado</u>
<br>Quebra de linha

Este é um parágrafo.

Título de Nível 1

Título de Nível 2

Texto em negrito
Texto em itálico
Texto sublinhado
Quebra de linha

<blockquote> e <pre>

Estas tags são usadas para citações em bloco e blocos de código pré-formatados.

<blockquote>Isto é uma citação.</blockquote>
<pre>
    <code>Código pré-formatado</code>
</pre>
Isto é uma citação.
Código pré-formatado

4. Listas

<ul>, <ol> e <li>

Cria listas não ordenadas (<ul>) e ordenadas (<ol>) com itens de lista (<li>).

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>
  • Item 1
  • Item 2
  1. Item 1
  2. Item 2

5. Links e Imagens

<a> e <img>

Cria links e insere imagens.

<a href="https://www.example.com">Visite o site</a>
<img src="imagem.jpg" alt="Descrição da imagem">
Visite o site

6. Tabelas

<table>, <tr>, <th>, <td>

Cria tabelas com linhas (<tr>), cabeçalhos (<th>) e células (<td>).

<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Celula 1</td>
        <td>Celula 2</td>
    </tr>
</table>
Cabeçalho 1 Cabeçalho 2
Celula 1 Celula 2

7. Elementos Sem Fechamento

<br>, <hr>, <img>, <meta>

Essas tags não possuem uma tag de fechamento e são usadas para quebra de linha, linha horizontal, imagens e metadados.

<br> 
<hr> 
<img src="imagem.jpg" alt="Imagem descritiva"> 
<meta name="description" content="Descrição da página">
[Exemplo: Quebra de linha e linha horizontal não visíveis aqui]

8. Elementos de Código

<code> e <pre>

Cria blocos de código e texto pré-formatado.

<code>document.write('Hello, World!');</code>
<pre>
    function hello() {
        console.log('Hello, World!');
    }
</pre>
document.write('Hello, World!'); 
    function hello() {
        console.log('Hello, World!');
